# Repository Guidelines

## 项目概述

这是基于Python + Pygame开发的2D横版平台跳跃游戏（类似超级马里奥）。玩家控制卡通角色通过2个关卡，每关末尾有Boss守门。

## 项目结构

```
mario2/
├── src/                    # 源代码目录
│   ├── main.py            # 游戏入口
│   ├── player.py          # 玩家角色
│   ├── enemy.py           # 敌人小怪
│   ├── boss.py            # Boss类
│   ├── level.py           # 关卡管理
│   ├── platform.py        # 平台/地形
│   ├── projectile.py      # 子弹/火球
│   ├── collision.py       # 碰撞检测
│   ├── sprites.py         # 精灵绘制（像素风）
│   └── constants.py       # 常量定义
├── assets/                 # 素材目录
└── AGENTS.md              # 贡献指南
```

## 运行命令

```bash
# 安装依赖
pip install pygame

# 运行游戏
python src/main.py
```

## 代码规范

- **缩进**: 4空格
- **命名**: `snake_case`（函数/变量），`PascalCase`（类）
- **类型提示**: 建议添加返回值类型注解

## 核心模块说明

| 模块 | 职责 |
|------|------|
| `player.py` | 玩家移动、跳跃、攻击逻辑 |
| `boss.py` | Boss AI和攻击模式 |
| `level.py` | 关卡数据与切换 |
| `sprites.py` | 像素艺术绘制 |

## 操作方式（游戏内）

- **方向键**: 左右移动
- **空格**: 跳跃
- **F**: 攻击

## 开发建议

- 先运行 `python src/main.py` 验证环境
- 修改常量在 `constants.py` 中调整
- 新增关卡在 `level.py` 中添加数据
